# Learning the letters of the alphabet

</h1>

The idea behind this mini-game is to make it easier for children of literacy age to memorize the letters of the alphabet in a more efficient, didactic, and "fun" way.

To achieve this, a simple Python code was developed so that, when letter keys are selected, the indicated letter appears with a specific window color. At this point, in order to facilitate learning, colors that attract children's attention were chosen, precisely to better focus their attention on learning and consequently make the game more attractive.

The ultimate goal of the project was to develop code that is easy for users to implement and access, as well as to develop practical application concepts in Python.
